Quick Start

• Have them think of an everyday object they're holding
• Tap Hangman to start
• Begin with E — ask if it's in their word

• If no → draw a hangman part
• If yes → write it below the gallows

• Location of each input advances anagram accordingly
• SOLVE = peek next letter (visible only while finger is on screen)
• Undo = remove last stroke

• Keep going until the word appears
• Tap SOLVE to reveal, Hangman to reset
• Double-tap top-right for Settings

Tip: I emphasize that, unlike normal hangman, I don't know the word length, letter positions, or repeats.
