## Installation

### Mobile (iOS/Android)
1. Open the site in your mobile browser
2. Tap the share/menu button
3. Select "Add to Home Screen"
4. Tap "Add"

### Desktop
Use your browser's install option from the address bar or menu.

## Development

This is a static site using vanilla HTML, CSS, and JavaScript.

### Files
- `index.html` - Main application page
- `script.js` - Application logic
- `base.css` - Styles and layout
- `manifest.json` - PWA configuration
